# KITE (AI Financial Intelligence System) 

AI-powered stock market intelligence platform with ML predictions,
real-time analytics & professional trading UI.

##  Tech Stack
- **Backend:** FastAPI, Python, scikit-learn, XGBoost, yfinance
- **Frontend:** Next.js 14, TypeScript, TailwindCSS, Chart.js

##  Features
- AI Stock Predictions (BUY/SELL/HOLD)
- Interactive Price Charts
- 3 ML Models: Linear Regression, Random Forest, XGBoost
- Dark/Light Mode
- Real-time Ticker Tape
- Simulated Trading Panel
